TSDKCore

public class TSDKCore : NSObject

Main telerion SDK class. Must be used as telerion SDK entry point and linked to telerion service via init(‘telerion-URL’) before any other actions. Any outgoing or incoming session requires to be authenticated so make authentication first via login(…). SDK keeps authentication while app is alive or till logout( ) is called.
